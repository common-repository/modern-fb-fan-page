== Description ==
IT is modern-fb-fan-page Plugin. It is simple plugin but useful. 

Some Of The Feature of plugin 

1. Dynamic Facebook Page URL
2. Customize Width and Height 
3. Use Small Header show or hide option
4.  Cove Photo hide or show option
5.  Show Friend's Faces hide or show option
6. Multiple Tabs option => 1. Show or hide timeline. 2. Show or hide Message 3. Show or hide events. 4. Nothing to show
